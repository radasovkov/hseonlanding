# HSE.onl Landing

Landing page en React para HSE.onl, un feed profesional en español donde la inteligencia artificial genera preguntas a partir de trayectorias y las personas comparten respuestas sobre decisiones, errores y aprendizajes de carrera.

## Incluye

- Hero con propuesta de valor y descarga directa desde App Store.
- Mockups del feed infinito, un hilo de respuestas y el editor de una respuesta.
- Flujo de producto basado en subir el currículum, ayudar a otras personas y reflexionar sobre los propios errores.
- Condiciones visibles de la suscripción mensual de 20 €.
- Contacto por correo, Instagram y enlace a App Store.
- Diseño responsive y estados de foco accesibles.

## Desarrollo

```bash
npm install
npm run dev
```

Para generar la versión de producción:

```bash
npm run build
```

## Enlaces de marca

El correo y los enlaces externos están centralizados en `brandLinks`, al principio de `src/App.jsx`. El enlace actual de App Store abre la búsqueda de HSE.onl; sustitúyelo por la URL directa de la ficha cuando se facilite su ID público.
